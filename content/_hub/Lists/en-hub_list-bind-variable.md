You can link individual cells in Peakboard Hub lists with Peakboard Hub variables.
This creates a bidirectional synchronization where changes to the variable automatically update the linked cell and vice versa.

![Link list cell to variable](/assets/images/hub/de_hub_list-bind.gif)

### Link cell to variable

1. **Select cell:** Click on the cell in your list that you want to link to a variable. The selected row will be highlighted.
2. **Create link:** Click [Bind to variable] in the upper toolbar. A dialog window with available variables opens.
3. **Select variable:** Choose the desired variable from the displayed list. For example, for a string column, only string variables are shown for selection.
4. **Confirm link:** After selection, the variable is successfully linked to the cell.

### Remove link

1. **Select linked cell:** Select the cell whose link you want to remove.
2. **Select unbind:** Click [Unbind] in the toolbar.
3. **Confirm removal:** A confirmation dialog appears. Click [Yes] to remove the link between the cell and the variable.
