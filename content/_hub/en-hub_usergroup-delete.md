In this article you will learn how to delete an existing user group in Peakboard Hub.
After completing the described steps, the selected group is irrevocably removed from the system.

![Delete user group](/assets/images/hub/de_hub_usergroup-delete.gif)

### Select group for deletion

1. **Select user group:** Navigate to [Users] in the left menu and select the user group you want to delete from the list. The selected group is highlighted in blue.
2. **Open context menu:** Click the three-dot icon to the right of the group name.

### Perform deletion

3. **Choose delete option:** Select the [Delete] option from the dropdown menu.
4. **Confirm deletion:** A confirmation dialog appears. Click the [Yes] button to confirm the deletion.

<div class="box-warning" markdown="1"> **Please note:**
This process cannot be undone. All users assigned to this group will lose the corresponding group permissions.
</div>
